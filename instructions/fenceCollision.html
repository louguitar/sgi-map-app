SGI fence collision layer visualization

The following will visualize the fence collision layer in ArcMap as it is shown in the SGI web application. Please note that the visual layer is for easy visualization only and is not suitable for analysis. Use the data layer for analysis and custom visualization.

  • Download the visualization TIF file as directed in the email
  • Add file to ArcMap using the Add Data button

  • Double click on the layer to change symbology

  • Check "Display Background Value"
  • Change "Stretch Type" to "None"
  • Uncheck "Apply Gamma Stretch"
  • Click "OK"
